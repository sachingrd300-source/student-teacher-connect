# **App Name**: EduConnect Pro

## Core Features:

- Role-Based Access: Secure access control for teachers, students, and parents.
- Teacher Registration & Management: Teachers can register, manage students, upload materials, and control class settings.
- Student Enrollment Modes: Two modes: Independent (basic access) and Teacher-Connected (full access upon teacher approval).
- Parent Enrollment & Read-Only Access: Parents enroll with teacher approval and have read-only access to student data.
- Study Material Module: Teachers upload notes, DPPs, tests, and solutions; supports subject-wise organization and offline access.
- Real-Time Updates & Notifications: Live UI updates and push notifications for new materials, attendance, and schedule changes.
- Performance & Attendance System: Subject-wise attendance and test marks with weekly/monthly performance graphs.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and stability.
- Background color: Light gray (#F5F5F5) for a clean and modern interface.
- Accent color: Vibrant orange (#FF9800) for important actions and highlights.
- Body font: 'PT Sans' (sans-serif) for readability and clarity.
- Headline font: 'Playfair' (serif) for titles to create a high-end feel. Use 'PT Sans' (sans-serif) for body if longer text is anticipated
- Use clean, professional icons representing subjects, modules, and actions.
- Subtle animations for loading data and real-time updates to enhance user experience.
